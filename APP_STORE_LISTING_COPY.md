# App Store Listing Copy — Reverse Bundle Pro

Paste into **Shopify Partners → Apps → Reverse Bundle Pro → App listing**.

Keep the name **Reverse Bundle Pro** (do not rename unless you rebrand everywhere).

---

## Short description (160 chars max)

```
Turn repeat multi-SKU orders into one bundle SKU before fulfillment. Cut pick steps and shipping cost for high-volume Shopify stores.
```

## Intro / hero paragraph

```
Reverse Bundle Pro groups incoming orders that match your bundle rules and converts them to a single pre-packed SKU—so your warehouse picks one item instead of many.

Built for Shopify merchants shipping 50+ orders/day with repeat product combinations (supplements, accessories, variety packs, subscription boxes).
```

## Who it's for

```
✓ Stores with multiple SKUs often ordered together
✓ Merchants using ShipStation or a 3PL
✓ Brands where pick-pack cost is a real line item
✓ Operations teams tired of manual bundle tagging
```

## What happens after install (plain steps)

```
1. Install and approve permissions
2. Complete the 5-minute setup wizard
3. Create your first bundle rule (or use AI suggestions)
4. New orders matching the rule are tagged or converted automatically
5. Track savings in the analytics dashboard
```

## Key features (bullet list)

```
• Rule-based reverse bundling — match orders to pre-defined bundle SKUs
• Setup wizard — first bundle rule in minutes
• Order tagging & optional order edits for fulfillment
• AI bundle suggestions from order history
• Analytics — conversions, savings, rule performance
• ShipStation-ready workflow via Shopify order tags
```

## Permissions note (add to listing FAQ)

```
We request order and product access to read incoming orders and apply bundle rules. Customer PII is not required for core bundling—we only use what Shopify provides for order processing. Uninstalling deletes your store data.
```

## Do NOT claim (trust killers)

- ❌ "30% cost reduction guaranteed"
- ❌ "Predict demand" / "enterprise AI"
- ❌ Fake reviews or developer self-replies
- ❌ SOC 2 certified (unless you actually are)
- ❌ Integrations you haven't shipped (FBA, EDI) as "operational"

## Review strategy

Ask for a review only after a merchant has:

1. Active rules for 2+ weeks
2. Measurable conversions in analytics
3. Agreed in writing to share their store name

Real review template to request:

```
"We've been using Reverse Bundle Pro for [X weeks]. It automated [specific SKU combo] and our team now picks [1 SKU vs 4]. Saved roughly [$ or time] per week."
```
